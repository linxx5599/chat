### server 服务端资源
 * node + express
### 进入项目目录 安装依赖 （推荐yarn）
```
cd server
npm install  / yarn
```
### 安装nodemon（推荐）
```
npm insatll nodemon -g
```
### 启动服务
```
npm run start  / yarn start
```
### node 启动
```
npm run dev  / yarn dev
```

### 打包
```
npm run build  / yarn build
```

### 本地预览打包查看依赖大小
```
npm run build:analy / yarn build:analy
```

### 本地预览打包后的dist文件
```
npm run build:pre / yarn build:pre
```
